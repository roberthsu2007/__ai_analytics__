# AI 數據分析與洞察工具 (AI Data Analysis and Insights Tool)

這是一個專業的 AI 數據分析工具，能夠快速將原始的 CSV 資料轉化為深入的商業洞察與摘要報告。本專案整合了 Google Gemini AI 模型，協助使用者在無需具備專業數據分析背景的情況下，也能輕鬆獲得數據背後的價值與趨勢。

## ✨ 主要功能 (Features)

- **CSV 數據解析**：支援貼上原始 CSV 格式資料進行即時處理。
- **Gemini AI 分析**：利用 Google Gemini 強大的語言與邏輯能力，自動產生數據趨勢報告。
- **Markdown 報告渲染**：分析結果以結構化的 Markdown 呈現，易於閱讀與重點擷取。
- **一鍵複製結果**：方便將分析洞察快速應用於簡報或報告中。
- **現代化 UI 設計**：具備深色/淺色模式，提供直覺且順暢的使用者體驗。

## 🛠️ 技術棧 (Tech Stack)

- **前端框架**: React 19, Vite
- **UI 樣式**: Tailwind CSS, Lucide React (Icons), Motion (動畫)
- **AI 整合**: Google Gemini API (`@google/genai`)
- **部署環境**: Vercel Serverless Functions
